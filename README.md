#  Image Slider

A smooth image slider built using HTML, CSS, and JavaScript.  
Features automatic and manual navigation between images.

##  Tech Used
- HTML
- CSS
- JavaScript

##  Features
- Auto-slide with timer
- Manual navigation (Next/Previous)
- Responsive layout
- Image preloading
